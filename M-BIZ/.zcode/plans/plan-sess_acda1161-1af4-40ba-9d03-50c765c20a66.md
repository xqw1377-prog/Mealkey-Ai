## 实现方案：接入 DashScope Paraformer 语音转写

### 目标
将 `asset.service.ts` 中空实现的 `transcribeAudio` 函数替换为真实调用阿里云 DashScope Paraformer-v2 语音识别 API，使上传的语音文件能自动转写成文字，再交给 DeepSeek 做经营摘要。

### 改动文件

#### 1. `apps/web/src/server/services/asset.service.ts` — 核心改动

**替换 `transcribeAudio` 函数（第 473-480 行）**：

```ts
async function transcribeAudio(file: File, buffer: Buffer, transcriptHint: string | null): Promise<string | null> {
  // 1. 优先用前端浏览器实时识别的 transcriptHint（省 API 调用）
  if (transcriptHint && transcriptHint.trim().length > 0) {
    return transcriptHint.trim();
  }

  // 2. 调用 DashScope Paraformer-v2 同步语音识别
  const apiKey = process.env.DASHSCOPE_API_KEY;
  if (!apiKey) return null;

  try {
    const audioBase64 = buffer.toString("base64");
    const format = detectAudioFormat(file.type, file.name);

    const response = await fetch(
      "https://dashscope.aliyuncs.com/api/v1/services/audio/asr/transcription",
      {
        method: "POST",
        headers: {
          "Content-Type": "application/json",
          Authorization: `Bearer ${apiKey}`,
        },
        body: JSON.stringify({
          model: "paraformer-v2",
          input: { audio: { data: audioBase64 } },
          parameters: {
            sample_rate: 16000,
            format,
            language_hints: ["zh", "en"],
          },
        }),
      }
    );

    if (!response.ok) {
      const errorText = await response.text();
      console.error("[asset] dashscope asr failed", response.status, errorText.slice(0, 500));
      return null;
    }

    const payload = (await response.json()) as {
      output?: { text?: string };
    };
    const text = payload.output?.text?.trim() ?? "";
    return text.length > 0 ? text : null;
  } catch (error) {
    console.error("[asset] dashscope asr exception", error);
    return null;
  }
}
```

**新增 `detectAudioFormat` 辅助函数**（从 MIME/文件名推断 DashScope 需要的格式参数）：
```ts
function detectAudioFormat(mimeType: string, fileName: string): string {
  const ext = path.extname(fileName).toLowerCase();
  if (mimeType.includes("webm") || ext === ".webm") return "webm";
  if (mimeType.includes("mp4") || ext === ".m4a" || ext === ".mp4") return "mp4";
  if (mimeType.includes("wav") || ext === ".wav") return "wav";
  if (mimeType.includes("mp3") || ext === ".mp3") return "mp3";
  return "wav"; // 默认
}
```

### 不改动的部分
- **图片识别**：保持 Tesseract OCR，不变
- **浏览器实时语音**（`advisor/page.tsx` 的 `webkitSpeechRecognition`）：不变，仍然作为 `transcriptHint` 优先使用
- **`summarizeAssetEvidence`**：不变，转写出的文字仍走 DeepSeek 做摘要
- **存储逻辑**：不变，音频仍存本地/S3，转写在内存中用 buffer 完成

### 调用链路（改后）
```
用户上传语音 → upload/route.ts → createAssetFromUpload
  → extractAssetInsights (kind=audio)
    → transcribeAudio:
        ① 有 transcriptHint → 直接返回（省 API 调用）
        ② 无 transcriptHint → DashScope Paraformer-v2 base64 直传 → 返回 text
    → summarizeAssetEvidence (DeepSeek) → 生成经营摘要
```
