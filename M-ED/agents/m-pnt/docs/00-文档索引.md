## 术语速查

| 术语 | 含义 |
|---|---|
| 五选 | 竞争位置 / 心智客户 / 场景 / 差异化 / 验证路径 |
| 六维 | 市场 / 竞争 / 客群 / 场景 / 资源 / 防御 |
| 三理论 | Ries / Trout / Ye 正交多视角 |
| theory_recommend | 理论层推荐等级 |
| decision_recommend | 最终决策推荐等级 |
| M-Solution | Situation→…→Decision 七段交付 |
| R1–R4 | 统一风险等级（无 R5） |

## 概念映射（五选 × 三理论）

| 产品五选 | Ries | Trout | Ye |
|---|---|---|---|
| 竞争位置 | 第一/品类位置 | 竞品空位 | 可被认知的现实位置 |
| 目标心智客户 | 是否过宽 | 第一联想归谁 | 谁买单谁传播 |
| 核心消费场景 | 是否冲淡第一 | 能否钉住差异 | 是否真、频、可记 |
| 差异化方向 | 是否领导资产 | 是否锋利区隔 | 是否可感知可交付 |
| 验证与强化 | 长期能否强化 | 模仿后是否仍在 | 最小动作能否验证 |
