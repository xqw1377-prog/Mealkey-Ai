# 六维模型规范

## 模型接口

```
SixDimensionModel
├── name: str                    # 模型名称
├── description: str             # 模型描述
├── dimensions: List[Dimension]  # 六个维度（按 id 排序）
│   ├── Dimension_1 (市场容量, weight=0.25)
│   │   ├── name: str
│   │   ├── description: str
│   │   ├── weight: float (0-1)
│   │   ├── score: float (0-5)
│   │   └── indicators: List[Indicator]  # 5 个指标
│   ├── Dimension_2 (竞争格局, weight=0.20)
│   ├── Dimension_3 (消费适配, weight=0.20)
│   ├── Dimension_4 (运营可行性, weight=0.15)
│   ├── Dimension_5 (品牌势能, weight=0.10)
│   └── Dimension_6 (环境适配, weight=0.10)
├── get_opportunity_score() -> float      # 机会评分
├── get_opportunity_level(score) -> str   # 机会等级
└── summary() -> dict                     # 模型摘要
```

## 评分标准

每个维度得分范围：**0.0 - 5.0**

| 得分 | 等级 | 说明 |
|------|------|------|
| 4.0 - 5.0 | 优秀 | 显著优势 |
| 3.0 - 3.9 | 良好 | 具备优势 |
| 2.0 - 2.9 | 一般 | 行业平均 |
| 1.0 - 1.9 | 较弱 | 存在短板 |
| 0.0 - 0.9 | 差 | 明显劣势 |

## Indicator 指标定义

每个指标包含：
- **name**: 指标名称
- **description**: 指标说明
- **value**: 原始值（float | None）
- **normalized_score**: 归一化得分 0-5（float | None）
- **weight**: 在维度内的权重（0-1）

### Indicator 方法

```python
indicator.get_weighted_score() -> float   # 获取加权得分 (score × weight)
indicator.to_dict() -> dict               # 序列化为字典
```

## Dimension 方法

```python
dimension.calculate_score() -> float      # 计算维度得分（指标加权平均）
dimension.get_level() -> ScoreLevel        # 获取得分等级
dimension.to_dict() -> dict               # 序列化为字典
```

## SixDimensionModel 方法

```python
model.get_dimension(dim_id) -> Dimension | None  # 根据编号获取维度
model.calculate_all() -> None                     # 计算所有维度得分
model.get_opportunity_score() -> float            # 机会评分 (加权)
model.get_opportunity_level(score) -> str         # 机会等级（静态方法）
model.summary() -> dict                           # 模型摘要
```
