# Agent 内部 Workflow 主链路（速查）

来源：专业模型 V1.0.1 + 三理论 Agent 矩阵 V1

```text
1. Situation 解析
2. Insight 生成 + 六维诊断
3. Position 多方案生成（3–5）
4. 三理论矩阵并行
     ├─ Ries
     ├─ Trout
     └─ Ye Maozhong
5. Cross-Fire（冲突 / 共识 / 淘汰）
6. 评分 + 红队（R1–R4）
7. Synthesis → decision_recommend
8. Validation（30天 / 90天）
9. Quality Check → Final Output
```

## 多方案生成链路

```text
基础约束 → 竞争切口 → 差异维度 → 候选方向 → 成立前提 → 差异性检查
```

质量约束：≥1 稳健型、≥1 进攻型、≥1 备选；不可文案换词；每方案可一句话讲清。

## 三理论后必须产出

- 三份 Theory View  
- Cross-Fire 结果  
- primary / secondary / reject（及可选 backup）
