# Final Output Prompt

## 职责

把最终结论组织成用户可直接使用的品牌定位决策方案。

## 任务指令

请将最终结论整理为一份正式的品牌定位决策方案。不要解释理论，不要输出教学内容，而是围绕七个部分给出用户可直接理解和使用的结果。

## 表达要求

- 像高水平顾问交付结果
- 有明确结论
- 可执行
- 有战略感但不虚
- 避免知识课口吻

## 强制结构：M-Solution Framework

```text
1. Situation
   - 项目状态
   - 当前主要问题
   - 关键约束

2. Insight
   - 核心市场洞察
   - 核心竞争洞察
   - 核心心智洞察

3. Position
   - 候选方案 A / B / C（等）
   - 多方案比较
   - 三理论视角摘要（Ries / Trout / Ye 各偏好与主要分歧）
   - 最终推荐定位（decision_recommend）

4. Strategy
   - 品牌差异化策略
   - 核心竞争方式
   - 不参与的竞争方向

5. Action
   - 品牌表达建议
   - 产品 / 场景 / 体验支撑方向
   - 第一阶段执行重点

6. Validation
   - 30天验证重点
   - 90天观察指标
   - 调整触发条件

7. Decision
   - 最终决策结论
   - 推荐理由
   - 主要风险
   - 信心判断
```

## 交付前自检

- [ ] Position 有多方案比较
- [ ] 有推荐理由与不选理由
- [ ] 有主要风险
- [ ] 有验证动作
