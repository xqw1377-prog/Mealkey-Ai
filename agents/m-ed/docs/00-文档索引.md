## 术语速查

| 术语 | 含义 |
|------|------|
| 股权结构设计 | 根据项目阶段、团队角色、贡献度推荐合理的股权分配方案 |
| 动态调整建议 | 基于实际贡献、里程碑达成等数据提出股权调整建议 |
| 场景模拟 | 模拟不同分配方案下的未来股权演变（多轮融资稀释等） |
| 合规检查 | 对股权方案进行 Vesting、IP 转让、税务、创始人保护等合规审查 |
| 文档生成 | 自动生成股权协议草案、董事会决议、股东协议等法律文档 |
| LLM 模式 | 配置了 OPENAI_API_KEY 或 ANTHROPIC_API_KEY，调用大模型生成方案 |
| 规则引擎退路 | 未配置 API Key 或 LLM 调用失败时，使用预设规则生成基础方案 |
| Vesting | 股权成熟机制，通常 4 年成熟 + 1 年 Cliff |
| Cliff | 最短服务期，满 Cliff 后才开始获得股权 |
| 期权池 | 预留用于未来员工激励的股份 |
| 等比稀释 | 融资时所有股东按比例同步稀释 |
| AB 股 | 同股不同权，创始人通过高投票权股保持控制权 |
| 一致行动协议 | 多个股东约定在重大事项上统一投票的协议 |
