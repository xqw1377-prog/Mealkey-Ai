## е…ій”®и®ҫи®ЎеҶізӯ–

1. **еӯҳеӮЁж— е…іеј•ж“Һ**: knowledge-engine е’Ң memory-engine йҖҡиҝҮжҺҘеҸЈи§ЈиҖҰпјҢдёҚдҫқиө– PrismaгҖӮе…·дҪ“еӯҳеӮЁе®һзҺ°еңЁ apps/web еұӮжіЁе…Ҙ
2. **з»ҹдёҖдә§е“Ғи·Ҝз”ұ**: йҖҡиҝҮ `streamAgentResponse()` з»ҹдёҖе…ҘеҸЈпјҢжҢүж„ҸеӣҫжЈҖжөӢеҲҶжөҒеҲ° M-PNTпјҲе“ҒзүҢжҲҳз•Ҙе·ҘдҪңеқҠпјү/ M-EDпјҲиӮЎжқғпјү/ M-BIZпјҲе•ҶдёҡжЁЎејҸпјү/ ChiefAgentпјҲйҖҡз”Ёз»ҸиҗҘеҲӨж–ӯпјү
3. **иҮӘеҠЁйҷҚзә§**: LLM еҲӨж–ӯй“ҫеӨұиҙҘж—¶иҮӘеҠЁйҷҚзә§еҲ° DefaultJudgmentChainпјҲ50+ з»ҸиҗҘи§„еҲҷпјүпјҢз”ЁжҲ·ж— ж„ҹзҹҘ
4. **Mission йҖҡдҝЎ**: Agent й—ҙдёҚзӣҙжҺҘ importпјҢйҖҡиҝҮ MissionRouter еҸ‘йҖҒ/жҺҘж”¶ MissionпјҲеҪ“еүҚдёәеҚ• Agent жү§иЎҢпјҢдёәжңӘжқҘеӨҡ Agent еҚҸдҪңйў„з•ҷпјү
5. **з»ҸиҗҘиө„дә§еҢ–**: Decision/Memory/Report дёҚжҳҜж•°жҚ®еә“иЎЁпјҢжҳҜз»ҸиҗҘиө„дә§гҖӮжҜҸж¬Ў Agent жү§иЎҢйғҪеҶҷе…Ҙ
