## 一、层级上限：3（冻结）

| 层 | 名字 | 一屏放什么 | 禁止 |
|----|------|------------|------|
| **L1** | 今日首屏 / 当前页主任务 | 问候 · 1 句状态 · **1 条主事项** · **1 个主 CTA** | 证据链全文、四维健康、收件箱列表、长假设、闭环轨道、充值条抢英雄位 |
| **L2** | 今日更多 / 页内展开 | 其他变化标题、收件箱数字、今日动作摘要、上传数据、账户、身份 | 七常委全文、档案全文 |
| **L3** | 独立页 | 经营分析 · 决策室 · 决策档案 · 账单 | 再嵌第四层「详情的详情」 |

**铁律：** 从 L1 到最终动作，点击深度 ≤ 3。

```text
L1 今日雷达（看一句 + 点一下）
  → L2 今日更多 / 或直达 L3 经营分析
  → L3 决策室 / 档案 / 打卡
```

## 二、今日页 = 主页面模块栏（冻结 · 2026-07-21）

**变化解读不进二级页**，全部在今日主页面用动态模块展示：

```text
今日经营（问候 + 一句摘要）

── 动态模块栏（有变化才充实，无则诚实空态）──
门店经营     数据卡 + 解读
舆情与外部   数据卡 + 解读
推进中       Agent / 执行事项进度
（可插更多工具 Agent 模块）

── 变化解读 · 今天最值得关注（本页主位）──
标题 + 判断 + 影响/建议 + 短证据
[ 带着这条进决策室 ]

[ 进入决策室 ]   ← 空白开案；**拍板只在决策室**
```

**拍板铁律：** 拍板 / 签字只在决策室。今日页禁止拍板按钮。  
**禁止：** 为「变化解读」再开独立二级页；账户余额上今日（→「我的」）。
