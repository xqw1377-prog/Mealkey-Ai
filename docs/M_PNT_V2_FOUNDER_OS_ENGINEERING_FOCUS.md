## ExpertReport 契约（喂常委）

Section IDs（与 `founder-os/expert-engines` 对齐）：

1. `category` — 品类 / 战场  
2. `mindset` — 用户心智 / 洞察  
3. `positioning` — For Who / Category / Difference / Reason  
4. `brand_strategy` — 品牌系统与执行主线  

常委消费镜头：CBO / CSO / CMO。

---

## Brand Strength（简化 Interbrand 维）

| 维 | 含义 |
|---|---|
| Clarity | 定位一句话可复述 |
| Differentiation | 相对竞品差异强度 |
| Consistency | Brand System 与合同一致 |
| Relevance | 用户需求匹配 |
| Evidence | 证据账本强度 |

输出：0–100 分 + 等级 + 缺口清单（供签字门禁与常委条件支持）。
